# TravelFreight_FrontEnd
Build a react native mobile app using Expo. 

Implemented login, sign up, join community, user profile, settings, and post ads screens with navigation flows.

Built backend using Node.js and Express, and Mongodb (hosted on heroku). Code located in a separate git repository [here](https://github.com/tongxu95/TravelFreight_Server). 

Implemented email validation, password policy (at least 8 characters), and uniqueness validation for sign up (unique email and username); used AsyncStorage to store user login credentials; and implemented address detection and profile image update functionalities.
